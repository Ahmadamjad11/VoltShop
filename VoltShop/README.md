# VoltShop - متجر الأجهزة الكهربائية

متجر إلكتروني متكامل للأجهزة الكهربائية والإلكترونية مع تصميم حديث وواجهة مستخدم سهلة.

## 🚀 المميزات

### الفرونت إند (Frontend)
- ✅ تصميم حديث ومتجاوب
- ✅ واجهة مستخدم سهلة ومتطورة
- ✅ نظام سلة تسوق متكامل
- ✅ صفحة دفع آمنة
- ✅ لوحة إدارة احترافية
- ✅ دعم اللغة العربية بالكامل

### الباك إند (Backend)
- ✅ API متكامل مع Express.js
- ✅ قاعدة بيانات MongoDB
- ✅ نظام إدارة المنتجات والفئات
- ✅ نظام إدارة الطلبات
- ✅ نظام مصادقة الأدمن
- ✅ بيانات تجريبية جاهزة

## 📦 التثبيت والتشغيل

### 1. تثبيت المتطلبات
```bash
# تثبيت Node.js (الإصدار 16 أو أحدث)
# تثبيت MongoDB
```

### 2. استنساخ المشروع
```bash
git clone <repository-url>
cd VoltShop
```

### 3. تثبيت التبعيات

#### الباك إند
```bash
cd backend
npm install
```

#### الفرونت إند
```bash
cd frontend
npm install
```

### 4. إعداد قاعدة البيانات
```bash
# تأكد من تشغيل MongoDB
# تشغيل البيانات التجريبية
cd backend
node seed.js
```

### 5. تشغيل التطبيق

#### الباك إند
```bash
cd backend
npm start
# سيعمل على http://localhost:5000
```

#### الفرونت إند
```bash
cd frontend
npm run dev
# سيعمل على http://localhost:5173
```

## 🔧 API Endpoints

### المنتجات
- `GET /api/products` - جلب جميع المنتجات
- `GET /api/products/:id` - جلب منتج محدد
- `POST /api/products` - إضافة منتج جديد
- `PUT /api/products/:id` - تعديل منتج
- `DELETE /api/products/:id` - حذف منتج

### الفئات
- `GET /api/categories` - جلب جميع الفئات
- `POST /api/categories` - إضافة فئة جديدة
- `PUT /api/categories/:id` - تعديل فئة
- `DELETE /api/categories/:id` - حذف فئة

### الطلبات
- `GET /api/orders` - جلب جميع الطلبات
- `POST /api/orders` - إضافة طلب جديد
- `PUT /api/orders/:id` - تحديث حالة الطلب

### الأدمن
- `POST /api/admin/login` - تسجيل دخول الأدمن
- `GET /api/admin/profile` - جلب بيانات الأدمن
